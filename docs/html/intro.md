# What is HTML?

HTML or Hyper Text Markup Language is how you make the page displayed by the browser.

## Basic skeleton

```html
<!DOCTYPE html>
<html>
  <head>
    <title>Page Title</title>
  </head>
  <body>
    <h1>This is a Heading</h1>
    <p>This is a paragraph.</p>
  </body>
</html>
```
